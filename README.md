# Vite Tutorial
Sing-a-long with "Learn Vite - Frontend Build Tool Course" by freeCodeCamp.org  
ref: https://www.youtube.com/watch?v=VAeRhmpcWEQ

# Setup
```shell
$> mkdir vite-prj
$> cd vite-prj
$> npm install --save-dev vite
$> git init .
```
Add scripts to the composer.json file:
```json
{
  "scripts": {
    "dev": "vite dev"
  }
}
```

# Dev Server
```shell
$> npm run dev
```
